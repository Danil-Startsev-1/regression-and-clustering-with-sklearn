# regression-and-clustering-with-sklearn
Репозиторий содержит учебный проект по задаче регрессии и кластеризации на реальном табличном датасете (Football Players Stats). В работе выполняются загрузка и предварительный анализ данных с помощью Pandas, обработка пропусков и искажений и стандартизация признаков.

Далее решается задача регрессии с использованием библиотеки sklearn: LinearRegression, полиномиальная регрессия (PolynomialFeatures), Lasso и Ridge Regression, а также RANSACRegressor. Для моделей считаются метрики MSE, MAE и R² на обучающей и проверочной выборках, строятся residuals plot и формулируются выводы. Отдельно применяется кластеризация методами KMeans и DBSCAN, анализируется связь кластеров с диапазонами признаков и целевой переменной, строятся scatter plot для выбранных признаков, а качество кластеризации оценивается с помощью силуэтного коэффициента и правила локтя.
